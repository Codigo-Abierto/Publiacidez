El proyecto Publiacidez nace una tarde de confinamiento de Mayo de 2020,
con el objetivo crear un bloqueador de anuncios y otros elementos molestos ligero, eficiente y con un código fuente abierto, sencillo y fácil de entender pero no por ello menos efectivo.

Aun siendo una extensión creada inicialmente para usuarios con un mínimo de conocimientos de programación, que se puedan beneficiar de la creación de sus propias reglas personalizadas de bloqueo y ocultación de elementos de las páginas web que más frecuentemente utilicen, puede resultar útil también para cualquier otro tipo de usuario con los dominios que vienen por defecto.
Lista que pretende incluir una selección de las webs más visitadas en diversas temáticas, y que se irá ampliando en sucesivas actualizaciones.

A diferencia de otros bloqueadores de anuncios, que están pensados para funcionar de forma global en todo internet, cargando decenas de miles de reglas de filtrado, con el consiguiente gasto de recursos que eso conlleva,
Publiacidez tiene un enfoque distinto, cargando las reglas de forma dinámica para los dominios que se hayan definido, en el momento en que el usuario accede a ellos.

Se permiten igualmente filtros permanentes, pero en esencia las reglas se definen por dominio, y se permite el bloqueo de peticiones a URLs, ocultación cosmética de elementos CSS dentro de la página y bloqueo selectivo de ejecución de scripts inline.

Adicionalmente, la extensión permite la deshabilitación global de ejecución de código JavaScript para cualquier página web (modo NoScript) e incluye otras opciones que facilitan la elaboración de las reglas de filtrado personalizadas.

El proyecto se continuará con una serie de videos explicativos del código fuente de la extensión y de su uso para creación de reglas personalizadas en una selección de casos prácticos representativos.
